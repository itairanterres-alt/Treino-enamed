# Régua qualitativa do revisor cego

O segundo modelo recebe a questão sem gabarito, justificativas ou pérola. Primeiro resolve o item; depois atribui uma nota de 0 a 2, com evidência textual concreta, em cinco dimensões.

| Dimensão | Pergunta de controle |
|---|---|
| Foco decisório | O item exige uma decisão clínica definida, em vez de mera lembrança isolada? |
| Suficiência dos dados | Há dados necessários e não há omissão que torne duas respostas defensáveis? |
| Plausibilidade dos distratores | As alternativas erradas são clinicamente relacionadas e discriminam raciocínio? |
| Ausência de pistas | Comprimento, linguagem, repetição ou precisão não denunciam o gabarito? |
| Segurança clínica | A resposta não depende de conduta potencialmente danosa ou desatualizada sem fonte confirmada? |

## Escala e gate

- `0`: falha grave ou risco; o item deve ser bloqueado ou reconstruído.
- `1`: problema reparável, com indicação objetiva do defeito.
- `2`: dimensão integralmente atendida, com evidência observável.

A verificação automática só aprova quando as cinco dimensões recebem 2, o revisor resolve a mesma alternativa armazenada, recomenda aprovação, considera o item seguro e o benchmark estrutural também passa. A fonte permanece `unverified` até conferência documental, e a questão permanece fora do banco humano até curadoria editorial.

Essa régua é um gate de redução de risco, não uma validação psicométrica nem substituto de revisão por docente especialista.
